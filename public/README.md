# Public Assets

Deze map bevat alle statische bestanden die direct toegankelijk zijn via de website.

## Logo

Plaats hier je logo bestand met de naam **`logo.png`**

### Aanbevolen specificaties:
- **Formaat**: PNG met transparante achtergrond
- **Grootte**: Minimaal 200px hoog (hogere resolutie voor retina displays)
- **Bestandsnaam**: `logo.png`
- **Kleur**: Het logo wordt weergegeven op een donkergroene achtergrond, dus zorg dat het goed zichtbaar is

### Alternatieve formaten:
Als je een ander formaat wilt gebruiken (bijv. SVG, JPG), pas dan ook het pad aan in:
- `app/page.tsx` (regel 63): wijzig `/logo.png` naar je gewenste bestandsnaam

## Andere bestanden

Je kunt hier ook andere statische bestanden plaatsen zoals:
- `favicon.ico` - Het favicon voor in de browser tab
- Afbeeldingen voor de foto placeholders
- Eventuele andere documenten of bestanden

Alle bestanden in deze map zijn toegankelijk via `http://jouwdomain.nl/bestandsnaam`
